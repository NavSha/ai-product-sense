# AI Product Sense — The New PM Interview Skill

An interactive Airtribe resource teaching aspiring PMs how to answer AI-era product interview questions: a 7-lens framework, an interactive "build a strong answer" exercise, and practice questions.

**Live:** https://navsha.github.io/ai-product-sense/

Static site — no build step. Open `index.html` locally, or deploy the folder as-is.

- `index.html` — markup
- `data.js` — all editable copy (lenses, phrases, scenarios, practice questions)
- `app.js` — renders content + wires interactions
- `styles.css` — design system
- `hero.webp`, `og-image.jpg`, `favicon.svg`, `fonts/` — assets
